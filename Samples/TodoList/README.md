# TodoList Sample

A basic todo list API demonstrating core Accordant patterns.

## What It Demonstrates

- Multi-entity state (users with todos)
- CRUD operations with relationships
- Error handling (404, 409 responses)
- Test generation with `InputSet`

## Running

```bash
cd TodoList.Tests
dotnet test
```
