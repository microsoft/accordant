# BankAccount Sample

A simple bank account API demonstrating basic Accordant concepts.

## What It Demonstrates

- Basic state modeling (account balance)
- Simple CRUD operations (CreateAccount, Deposit, Withdraw, GetBalance)
- Validation logic (overdraft protection)
- Getting started with Accordant

## Running

```bash
cd BankAccount.Api.Tests
dotnet test
```
